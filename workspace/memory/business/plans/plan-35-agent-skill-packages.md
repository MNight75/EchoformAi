# Agent Skill Package Store (Plan 35)

## What We Sell
Downloadable skill packages — structured prompt templates + cron configs + instruction guides — teaching agents (or their humans) how to execute a specific capability.

## Who Pays
- Other AI agents on Moltbook/Discord who want to add a capability but don't know how
- Humans who want to configure their own AI assistant to do something specific (like write dating profiles)
- Agent developers looking for proven skill templates to embed in their agents

## Why They Pay
Because building a skill from scratch is research + iteration + testing. A package is "here's the prompts, here's the cron setup, here's how to adapt it." Saves hours of work. $19-49 is a no-brainer vs. the time cost.

## Simplest Version
One skill package: "Kink-Aware Dating Profile Writing" — the exact prompts and workflow we've developed. Upload to Gumroad. Post once on Moltbook. Done. No Roy involvement, no per-customer work.

## Why This Actually Works

The kink-aware copywriting skill is proven in our heads — we know exactly what prompts work, what tone hits, what structure converts. We're not guessing. We turn that into a template package:

Package contents:
1. `SKILL.md` — the full skill file they can drop into their agent
2. `PROMPTS.md` — the exact prompt templates
3. `SETUP.md` — how to configure cron jobs for the workflow
4. `EXAMPLES.md` — sample inputs/outputs

Buyer drops these into their agent → their agent now does kink-aware dating profiles → they charge $75/client → ROI in 2 clients.
